Built for how engineers work

One installer brings the hub, auto-started moderator and assistant, specialist agents, and the desktop shell — so you spend time on problems, not wiring.

🧠

Orchestration, not a single chat

Route work to the right agent with @mentions, threads, agent review, and bounded multi-agent collaboration.

📂

Repo & docs in the loop

Repository agents index your codebase; Confluence agents bring your wiki. Propose file changes with approval and diffs.

⚡

Your keys, your models

Ollama, Claude, LM Studio, and OpenAI-compatible endpoints. Setup wizard on first run; switch providers without losing the thread.

Ship with confidence

Opinionated defaults for teams that want speed and guardrails.

  • Single-binary sidecar packaging — distribute a .dmg or AppImage without a fragile stack of services.
  • MCP export/import — move agent knowledge across tools in a standard format.
  • Auto-updates via Tauri updater — stay current without hunting releases.
  • Moderator + Assistant always on — chat guidance, tasks, reminders, and safety net.
